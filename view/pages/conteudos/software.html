Software

O Software é um conjunto de instruções que podem ser feitas com qualquer linguagem de programação e inseridas na máquina, para que a máquina possa trabalhar e realizar/executar ações específicas. Os componentes lógicos de um computador, programas de rotina que controlam as funções da máquina são denominados de softwares. O hardware sem o software não tem funcionalidade.

Classificação do software
  • Software de Sistema
  • Software de Programação
  • Software de Aplicação

Software de Sistema

Ajuda trabalhar com hardware de computador e inclui:

  • Sistema Operativo

É o programa que controla o computador, sem o sistema operativo o PC é inutil. É o programa que permite ao utilizador interagir com o PC.

  • Controladores de Dispositivos

Um contralador de dispositivo(driver) é um programa de computador que permite programa de nivel superior(como sistema operativo ou aplicativos) se comunique com um dispositivo de hardware. O controlador de dispositivo depende do hardware e é especifico para cada sistema operativo.

  • Utilitários

São programas do computador criada para gerir ou melhorar o desepnho do hardware.


Software de Programação

Fornecem ferramentas que auxiliam programadores a escreverem programas de computadores inclui:

  • Compiladores

São programas de PC que transformam o codígo fonte( programas escritas numa linguagem de programção) em codígo objecto, que geralmente tem a forma binario.

  • Depuradores

São programas de computadores que testam, procuram e fazem sugestão para reduzir os erros de outros programa de PC.

  • Editores de Texto para linguagens de programação

São ficheiros ou programas usado para editar e digitar textos ou codígo de um programa.
  • NotPad++
  • Sublime Text
  • Bloco de Nota

Software de Aplicação

Permite que o utilizador realiza tarefas mais especifica(não relacionada com desenvolvimento informática) inclui:

  • Softwares corporativos
  • Jogos de PC
  • Softwares Educacionas
  • Softwares Medicionas
  • Softwares de entretenimentos
